Classless Design System

Qu'est ce que c'est ?

Il s'agit d'un design system se basant uniquement sur les balises HTML semantiques. Aucun usage de class= ou style= n'est requis

Pourquoi ?

Nous voyons à cette approche plusieurs avantages :

  • Reposer sur les balises HTML sémantiques nous aide à rendre notre site accessible
  • Ne pas devoir faire de CSS additionnel permet d'accélerer les développements
  • Ne pas devoir faire de CSS additionnel permet de s'assurrer de la cohérence globale du design
  • Se limiter aux balises HTML existantes encourage une certaine sobriété dans la conception des interfaces
